# README

## Overview

This RO-Crate collection was **regenerated on April 27, 2026** to improve consistency and metadata quality.

## Scope and Generation Policy

### Repository

RO-Crates for **SSBD:repository** are generated **only for entries that contain unique or original data not represented in SSBD:database**.

For entries that have richer and more complete metadata in **SSBD:database**, users are encouraged to refer to the corresponding **SSBD:database records** instead of the repository-derived RO-Crates.

### Database

RO-Crates for **SSBD:database** are generated based on **SSBD metadata versions v1, v2, and v3**.

Only records that include an associated **image dataset** are included in this RO-Crate generation.

## Notes

- This regeneration aims to clarify the separation of roles between *SSBD:repository* and *SSBD:database*.
- Users should preferentially use SSBD:database as the authoritative source when richer metadata is available.
- SSBD:repository RO-Crates are intended to complement, not duplicate, database records.
